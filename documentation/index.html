Getting Started

Welcome to Bostami - Tailwind CSS Personal Portfolio HTML5 Template We would like to thank you for choosing our template - Bostami.

It’s built with Tailwind CSS. It’s fully responsive and looks well. It is a modern design and high-speed performance ailwind CSS template. You can use it for Personal Portfolio Landing, Minimal Portfolio, Freelancer Portfolio, Designer Portfolio.

Requirements

There are system requirements in order to install and setup this template and its components properly. Make sure that you are running the node.js & npm install.

Recommended node and npm 

                        
                            node install
                            npm install

NPM : You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.
